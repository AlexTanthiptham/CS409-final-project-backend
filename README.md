# FinalProject-Backend
This repository contains the backend APIs for the CS409 Final Project, LlamaBear. LlamaBear is a website where users can upload their resumes for review by experts and peers. The backend is hosted on Vercel at https://finalproject-backend.vercel.app/.

The frontend code can be found at: https://gitlab.com/brendaaurelie/finalProject.
